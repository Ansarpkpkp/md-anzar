🚀Projects

🎫GoQRide (QR Based Ticket Booking System for Public Transport)

GoQRide is a paperless and self-ticket booking system with a unique bus QR code for every bus. With this system, users can book their tickets by scanning the QR code and conductors can track the number of passengers traveling in the bus. The benefit for the bus owners is that they can monitor the bus's performance, collection, and number of passengers instantly. 📲

🛡️Phishguard (Website for Checking Fake News and Phishing Websites using ML)

Phishguard is a website dedicated to checking fake news and phishing links. It helps users verify the credibility of news sources and identify potential online threats swiftly. Through advanced algorithms and user input, it assesses the authenticity of URLs and news articles, aiding in promoting digital literacy and safeguarding against misinformation and cyber threats. 🔍

🏥Doclive (Clinic Management System using Java Core, JSP, and Servlets)

Doclive is a comprehensive clinic management system designed to streamline operations in healthcare facilities. It enables efficient appointment scheduling, patient record management, billing, and doctor-patient interactions. Built using Java Core, JSP, and Servlets, it ensures secure data handling and seamless integration for an enhanced user experience. 📊
